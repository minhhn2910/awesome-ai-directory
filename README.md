## 🧑‍💻 Users

### 💬 General Assistants

- 🤖 **ChatGPT** — General assistant for writing, coding, analysis, and multimodal tasks. (**🆓/💲**) ([website](https://chatgpt.com))
- 🟦 **Claude** — Long‑context assistant great for drafting, reasoning, and document workflows. (**🆓/💲**) ([website](https://claude.ai))
- 🔎 **Gemini** — Google assistant connected to Search and Workspace workflows. (**🆓/💲**) ([website](https://gemini.google.com))
- 🐳 **DeepSeek Chat** — Fast, capable Chinese + English chat assistant with strong coding/reasoning. (**🆓**) ([website](https://chat.deepseek.com), [company](https://www.deepseek.com/en/))
- 🟣 **Qwen Chat** — Alibaba’s Qwen assistant for chat, docs, and multimodal tasks. (**🆓**) ([website](https://qwen.ai/qwenchat))
- 🟡 **Kimi** — Moonshot AI’s assistant focused on long documents, research, and office tasks. (**🆓/💲**) ([website](https://www.kimi.com/en), [Moonshot](https://www.moonshot.ai/))

### 🔎 Search & Research

- 🧾 **Perplexity** — Answer engine that cites sources and supports research‑style workflows. (**🆓/💲**) ([website](https://www.perplexity.ai))
- 📚 **Elicit** — Research assistant for literature review, paper discovery, and evidence extraction. (**🆓/💲**) ([website](https://elicit.com))
- 🧠 **Consensus** — Research search that summarizes and cites scientific findings. (**🆓/💲**) ([website](https://consensus.app))

### ⚡ General Automation & Productivity

- ⚡ **Zapier** — Connect apps and build AI-powered workflows and agents with minimal setup. (**🆓/💲**) ([website](https://zapier.com))
- 🧩 **Make** — Visual no-code builder to orchestrate multi-app automations and AI agents. (**🆓/💲**) ([website](https://www.make.com/en))
- 🧩 **n8n** — Self-hostable workflow automation with native AI steps and 400+ integrations. (**🧩🆓/💲**) ([website](https://n8n.io), [github](https://github.com/n8n-io/n8n))
- 🔘 **IFTTT** — Simple app/device automations (“if this then that”) for personal productivity. (**🆓/💲**) ([website](https://ifttt.com))
- 🦞 **OpenClaw** — Self-hosted AI assistant that can run persistent daily automations across your apps (use sandboxed setup). (**🧩🆓**) ([website](https://openclaw.ai), [github](https://github.com/openclaw/openclaw))

### 🧑‍💻 Coding

- 🐙 **GitHub Copilot** — In‑IDE code completion and chat assistant for developers. (**💲**, trial) ([website](https://github.com/features/copilot))
- 🧠 **Cursor** — AI‑native code editor for multi‑file refactors and agentic edits. (**🆓/💲**) ([website](https://cursor.com), [pricing](https://cursor.com/pricing))
- 🌬️ **Windsurf** — Agentic IDE with “Cascade” for autonomous multi‑step edits. (**🆓/💲**) ([website](https://windsurf.com), [pricing](https://windsurf.com/pricing))
- 🧩 **Codeium** — Coding assistant with broad IDE support and a strong free individual offering. (**🆓/💲**) ([website](https://codeium.com))

### 🖼️ Image & Video

- 🎨 **Midjourney** — High‑quality stylized image generation. (**💲**) ([website](https://www.midjourney.com))
- 🧩 **Stable Diffusion (ecosystem)** — Open image generation models with many UIs and workflows. (**🧩🆓**) ([website](https://stability.ai), [github](https://github.com/Stability-AI/stablediffusion))
- 🧠 **ComfyUI** — Node‑based workflow builder for Stable Diffusion pipelines. (**🧩🆓**) ([website](https://comfyui.org), [github](https://github.com/comfyanonymous/ComfyUI))
- 🎬 **Runway** — Generative video tools for editing and text‑to‑video. (**🆓/💲**) ([website](https://runwayml.com))
- 🟧 **CapCut** — Popular video editor with AI features (captions, effects, templates). (**🆓/💲**) ([website](https://www.capcut.com))
- 🧑‍🏫 **Synthesia** — Create presenter‑style videos from scripts and templates. (**💲**) ([website](https://www.synthesia.io))

### 🗓️ Productivity & Meetings

- 🪟 **Microsoft Copilot (M365)** — AI inside Office for drafting, summarizing, and analysis. (**💲**) ([website](https://www.microsoft.com/microsoft-copilot))
- 🟩 **Gemini for Workspace** — AI in Gmail/Docs/Sheets/Slides for writing and insights. (**💲**) ([website](https://workspace.google.com/solutions/ai/))
- 🧠 **Notion AI** — AI features for notes, docs, and databases inside Notion. (**🆓/💲**) ([website](https://www.notion.so/product/ai))
- 🗣️ **Fireflies.ai** — Meeting recording, transcription, and searchable summaries. (**🆓/💲**) ([website](https://fireflies.ai))
- 🧾 **Otter.ai** — Transcription and meeting summaries with speaker attribution. (**🆓/💲**) ([website](https://otter.ai))
- 🧑‍💼 **Grammarly** — Writing assistant for clarity and tone across apps and browsers. (**🆓/💲**) ([website](https://www.grammarly.com))

## 🛠️ Builders

### 🔑 APIs & Model Gateways

- 🧭 **OpenRouter** — One API to access many LLM providers/models with routing and unified billing. ([website](https://openrouter.ai), [docs](https://openrouter.ai/docs/api/reference/overview))
- 🌙 **Moonshot AI (Kimi) Open Platform** — Kimi API with long context and tool calling for building apps in the Moonshot ecosystem. ([website](https://platform.moonshot.cn/), [docs](https://platform.moonshot.cn/docs/guide/start-using-kimi-api))
- ☁️ **Together AI** — Full-stack platform offering model APIs plus fine-tuning and GPU compute. ([website](https://www.together.ai/), [fine-tuning](https://www.together.ai/fine-tuning))
- 🎆 **Fireworks AI** — Model API + managed fine-tuning for popular open model families. ([website](https://fireworks.ai), [fine-tuning](https://fireworks.ai/blog/fine-tune-launch))

### 🧠 Agentic & GenAI App Frameworks

- 🧱 **LangChain** — Build LLM apps with tool calling, RAG, memory, and integrations. ([website](https://www.langchain.com), [github](https://github.com/langchain-ai/langchain))
- 🧩 **LangGraph** — Stateful, controllable agent workflows as graphs (great for production agents). ([website](https://langchain-ai.github.io/langgraph/), [github](https://github.com/langchain-ai/langgraph))
- 🗂️ **LlamaIndex** — Data framework for RAG over documents, databases, and APIs. ([website](https://www.llamaindex.ai), [github](https://github.com/run-llama/llama_index))
- 🧑‍🤝‍🧑 **AutoGen** — Multi‑agent framework for collaborative tool‑using agents. ([website](https://microsoft.github.io/autogen/), [github](https://github.com/microsoft/autogen))
- 🧭 **crewAI** — Orchestrate role‑based agent “crews” for multi‑step tasks. ([website](https://www.crewai.com), [github](https://github.com/crewAIInc/crewAI))

### 🧪 Models · Training · Hubs

- 🤗 **Hugging Face Hub** — Central hub for models, datasets, and Spaces used across the GenAI ecosystem. ([website](https://huggingface.co))
- 🤗 **Transformers** — Standard library for using and fine-tuning pretrained transformer models. ([website](https://huggingface.co/docs/transformers), [github](https://github.com/huggingface/transformers))
- 🟧 **PyTorch** — Most common deep learning framework for training and fine-tuning GenAI models. ([website](https://pytorch.org), [github](https://github.com/pytorch/pytorch))
- 🔶 **TensorFlow** — Google’s end-to-end framework for training and deploying models at scale. ([website](https://www.tensorflow.org), [github](https://github.com/tensorflow/tensorflow))

**Popular fine-tuning / training services:**
- 🤗 **Hugging Face AutoTrain** — No/low-code service to train and deploy custom models from your data. ([website](https://huggingface.co/autotrain))
- ☁️ **Together Fine-Tuning** — Managed fine-tuning for open models with eval tooling and production deployment paths. ([website](https://www.together.ai/fine-tuning))

### 🚀 Serving & Local Runtimes

- 🏃 **vLLM** — High‑throughput, memory‑efficient LLM serving engine. ([website](https://docs.vllm.ai), [github](https://github.com/vllm-project/vllm))
- 🤗 **HuggingFace** — For Hugging Face Model Serving Service. ([website](https://huggingface.co))
- 🧩 **llama.cpp** — Lightweight C/C++ inference for LLMs on CPU and consumer GPUs. ([website](https://ggml.ai), [github](https://github.com/ggerganov/llama.cpp))
- 🦙 **Ollama** — Simple local model runner + OpenAI‑compatible API for apps. ([website](https://ollama.com), [github](https://github.com/ollama/ollama))

### 📏 Evaluation & Observability

- 🧪 **Ragas** — RAG evaluation (retrieval quality, faithfulness, answer relevance). ([website](https://docs.ragas.io), [github](https://github.com/explodinggradients/ragas))
- 🧯 **promptfoo** — Prompt/model evals with CI‑friendly configs and comparisons. ([website](https://www.promptfoo.dev), [github](https://github.com/promptfoo/promptfoo))
- 🔭 **LangSmith** — Tracing + evaluation for LangChain/LangGraph apps. ([website](https://smith.langchain.com), [github](https://github.com/langchain-ai/langsmith-sdk))
- 📈 **Phoenix (Arize)** — Open‑source observability for LLMs, RAG, and embeddings. ([website](https://phoenix.arize.com), [github](https://github.com/Arize-ai/phoenix))
- 🛰️ **Helicone** — LLM observability proxy with logging, dashboards, and caching. ([website](https://www.helicone.ai), [github](https://github.com/Helicone/helicone))

### 🔌 Protocols & Tool Standards

- 🔗 **Model Context Protocol (MCP)** — Standard to connect LLM hosts to external tools/data via MCP servers. ([website](https://modelcontextprotocol.io), [github](https://github.com/modelcontextprotocol/modelcontextprotocol))
- 🤝 **Agent2Agent (A2A)** — Protocol for agents to interoperate across frameworks/vendors. ([website](https://a2a-protocol.org/latest/), [github](https://github.com/a2aproject/A2A))
